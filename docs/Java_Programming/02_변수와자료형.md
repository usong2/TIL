<h1>변수와 자료형</h1>

<h2>컴퓨터에서 자료 표현하기</h2>

<blockquote>이진수</blockquote>

<ul>
    <li>비트(bit) : 컴퓨터에서 자료를 표현하는 최소 단위 0 또는 1</li>
    <li>바이트(byte) : 8 bit</li>
    <li>이진수 : 컴퓨터에서 표현되는 자료의 형태 / 0000 1010 (이진수) => 10 (십진수)
    </li>
    <li>18진수 : 이진수를 3비트씩 모아서 표현</li>
    <li>16진수 : 이진수를 4비트씩 모아서 표현</li>
</ul>
