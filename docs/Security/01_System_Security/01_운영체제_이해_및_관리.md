## Point 1. CPU(Central Processing Unit, 중앙처리장치)의 구조적 이해

### 1. CPU의 구조 이해

#### (1) CPU 개념

입력장치로부터 자료를 받아서 연산하고 그 결과를 출력장치로 보내는 일련의 과정을 제어하며 해당 장치로 컴퓨터의 두뇌에 해당한다.

> **컴퓨터 시스템 구조**
>
> Main Memory ↔ ALU(CA)   ↔ 입출력장치(I/O)
>
> ​                                  ↑
> ​                                  ↓ Register
>
> 메모리(M) ↔ Control Unit(CC)
